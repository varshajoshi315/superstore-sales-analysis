## About the Project

This project is based on the Superstore sales dataset. The objective was to analyze the data using SQL and create an interactive dashboard in Power BI.

## Tools Used

- SQL (MySQL Workbench)
- Power BI
- Microsoft Excel
- GitHub

## Dashboard

The dashboard includes:
- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Sales by Region
- Profit by Category
- Monthly Sales Trend
- Top 10 Customers
- Filters for Region, Category and Month
